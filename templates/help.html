Documentação Completa BravON

BravON é uma linguagem de programação educacional projetada para gerenciamento de turmas e estudantes. Oferece uma sintaxe simples em português e um conjunto de comandos poderosos para operações académicas comuns.

Primeiros Passos

1. Acessar o Editor

Clique no botão "Editor" no menu para aceder ao editor de código BravON. Verá:

  • Área de Código - Digite seus comandos
  • Terminal (lado inferior) - Resultados e mensagens
  • Exemplos de coamndos - pode copiar clicando no no botão ao lado

2. Executar Código

Após digitar um comando, clique no botão "Executar" ou pressione Ctrl+Enter (se estiver no computador).

3. Primeiro Programa

CADASTRAR ESTUDANTE  "Nelma Bravo"  107 "10º A"
    CONSULTAR NOTAS 101
    LISTAR

Comandos Básicos

CADASTRAR ESTUDANTE

Propósito: Adicionar um novo estudante

Sintaxe:

CADASTRAR ESTUDANTE "<nome>"  <matrícula>  "<turma>"

Exemplo:

CADASTRAR ESTUDANTE "Nelma Bravo" 101  "10º A" 
CADASTRAR ESTUDANTE "Zulmira Milagre"  102  "10º A"

REMOVER ESTUDANTE

Propósito: Deletar um estudante

REMOVER ESTUDANTE 101

CONSULTAR

Propósito: Ver informações de um estudante

CONSULTAR NOTAS  101

ADICIONAR NOTAS

Propósito: Registrar notas para um estudante

ADICIONAR NOTAS 101 8.5 9.0 7.5

LISTAR TODOS

Propósito: Ver dados de todos os estudantes, não importa a turma

Comandos Avançados

PROMOVER TURMA

Propósito: Mover todos os estudantes de uma turma para outra

PROMOVER TURMA "10º A" "11º A"

PROMOVER ESTUDANTE NOVO

Propósito: Mover um estudante específico para outra turma

PROMOVER ESTUDANTE 101 "11º A"

RELATORIO

Propósito: Gerar relatório com estatísticas

RELATORIO
    RELATORIO "10º A"

SAIR

Propósito: Limpar todos os dados (IRREVERSÍVEL)

SAIR
CUIDADO: Esta operação não pode ser desfeita!

Exemplos Práticos

Exemplo 1: Criar uma Turma Completa

CADASTRAR ESTUDANTE "Nelma Bravo" 101  "10º A"

PROMOVER TURMA "10º A" "11º A"
    PROMOVER ESTUDANTE 105 "11º B"

FAQ & Troubleshooting

1. O que significa "Sintaxe inválida"?

R: Verifique se os comandos estão corretos, strings entre aspas e números sem aspas.

2. Os dados são salvos automaticamente?

R: Sim! Todo comando que modifica dados salva automaticamente em JSON.

3. Posso desfazer uma operação?

R: Não há undo integrado. Recomenda-se fazer backups periodicamente.

4. Qual é a matrícula máxima?

R: Qualquer número inteiro positivo até 2.147.483.647.
